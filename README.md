## Возможности

- Отображение поглощения на рейд-фреймах поверх полосы здоровья и входящего исцеления
- Чтение значения через `UnitGetTotalAbsorbs(unit)` и `VUHDO_getAbsorbOnUnit()`
- Отдельные цвета abs-бара при полном и неполном HP, а также при переполнении щита
- Раздельное отображение входящего исцеления: свои и чужие полосы
- Опциональная подсветка переполнения abs-бара (over-absorb glow)
- Выбор текстуры заливки abs-бара для полного и неполного HP
- Настройка интервала опроса поглощений (500–5000 мс)
- Автовключение отображения поглощений на Sirus при наличии `UnitGetTotalAbsorbs`
- Полезно ДЦ-жрецам для мониторинга эффектов `Слово Силы: Щит`, дополнительного поглощения от таланта `Божественное покровительство`, а также других эффектов поглощения

## Требования

- Клиент [Sirus.su](https://sirus.su) (Wrath 3.3.5)
- Основа: **VuhDo 2.23**
- Не устанавливайте параллельно стандартный VuhDo с Curse/Wago — используйте только эту версию

## Установка

1. Скачайте архив из [Releases](https://github.com/danilEl2/Vuhdo_Sirus.su/releases) (pre-release `ver. 2.23.4` или новее).
2. Распакуйте папки `VuhDo` и `VuhDoOptions` в `World of Warcraft\Interface\AddOns\`.
3. Если установлен другой VuhDo — удалите или переименуйте его папки.
4. Перезагрузите интерфейс: `/reload`.

## Настройка

1. Откройте настройки VuhDo: `/vd`.
2. Включите отображение: **Общее → Входящее → «Поглощение»** (по умолчанию выключено).
3. Задайте текстуры abs-бара: **Общее → Входящее → «Настройки поглощений»** — «Поглощения (неполное HP)» и «Поглощения (полное HP)».
4. При необходимости настройте интервал опроса поглощений в том же разделе (рекомендуется 3000–5000 мс).
5. Настройте цвета: **Цвета → Режимы → «Абсорбы»** — отдельно для неполного и полного HP.

## История изменений

### ver. 2.23.4 beta (по сравнению с 2.23.3 beta)

- Исправлено: abs-бар рисуется после входящего исцеления (как у Blizzard), без «вставки» между HP и хилом
- Исправлено: артефакты на стыках полос HP, входящего исцеления и поглощения (overlap и epsilon-текстуры)
- Исправлено: рамка подсветки цели/кластера скрывается при активном входящем исцелении
- Добавлено: раздельные полосы входящего исцеления — свои и чужие (**Общее → Входящее → «Раздельный»**)
- Добавлено: цвет поглощения при переполнении щита (**Цвета → Режимы → «Поглощения (переполнение)»**)
- Добавлено: опция подсветки переполнения abs-бара (**Общее → Входящее → «Переполнение»**)
- Добавлено: цвет входящего исцеления от других игроков в **Цвета → Режимы**
- Улучшено: fallback-проверка состава группы/рейда каждые 10 с для синхронизации фреймов

### ver. 2.23.3 beta (по сравнению с 2.23.2 beta)

- Исправлено: abs-бар больше не «прыгает» к центру фрейма при большом входящем исцелении
- Исправлено: HoT-бары учитывают прозрачность при проверке дистанции (затемнение вне радиуса)
- Добавлено: слайдер интервала опроса поглощений в **Общее → Входящее → Настройки поглощений**
- Улучшено: на Sirus поглощения включаются автоматически при первом запуске, если доступен `UnitGetTotalAbsorbs`
- Улучшено: интервал опроса поглощений берётся из настроек, без жёсткого fallback 5000 мс
- Мелкие правки локализации и вёрстки панели настроек поглощений
